Student Registration System
Mid-Semester Project – Back-End Web Development

Project Description
This is a PHP-based student portal that allows students to register, log in, select courses , upload a passport photo , and view generated results. The system uses session management and does not use a database.

Features
User Registration
Login & Logout (Session Management)
Course Registration
Profile
Dashboard Display
Index
Technologies Used
PHP
HTML5
CSS3
Sessions (No Database)
